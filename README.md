# CyberRent 🖥️

**Solusi Digital Warnet Modern**

CyberRent adalah aplikasi mobile yang dirancang untuk memberikan pengalaman manajemen warnet (internet cafe) yang modern dan efisien. Aplikasi ini memungkinkan pelanggan untuk menyewa PC, memesan snack, dan melacak riwayat transaksi mereka secara real-time.

## 📱 Tentang Aplikasi

CyberRent menghadirkan solusi digital lengkap untuk industri warnet dengan antarmuka yang user-friendly dan sistem manajemen yang terintegrasi. Aplikasi ini dibangun dengan teknologi terkini untuk memastikan pengalaman pengguna yang seamless.

**Status:** Proyek UAS (Ujian Akhir Semester) - Telkom University

## 🛠️ Teknologi yang Digunakan

- **Framework:** Flutter (Dart)
- **Backend & Database:** Firebase
  - Authentication (Firebase Auth)
  - Firestore (Real-time Database)
- **Bahasa Pemrograman Utama:** Dart
- **Platform Pendukung:** 
  - Android (Kotlin)
  - iOS (Swift)
  - Web (HTML)

## ✨ Fitur Utama

- 🖥️ **Penyewaan PC** - Sistem booking dan penyewaan PC dengan manajemen waktu real-time
- 🍕 **Pemesanan Snack** - Integrasi pemesanan makanan ringan dan minuman
- 💳 **Riwayat Transaksi** - Pencatatan lengkap dan real-time dari semua transaksi
- 🔐 **Autentikasi Aman** - Login dan registrasi terintegrasi dengan Firebase Authentication
- 💾 **Sinkronisasi Cloud** - Data tersimpan aman di Firestore dengan sinkronisasi real-time

## 🚀 Cara Memulai

### Prasyarat
- Flutter SDK (versi terbaru)
- Dart SDK
- Android Studio atau Xcode (untuk emulator)
- Firebase Project

### Instalasi

1. **Clone Repository**
   ```bash
   git clone https://github.com/MFaridIrham/CyberRent.git
   cd CyberRent
   ```

2. **Install Dependencies**
   ```bash
   flutter pub get
   ```

3. **Konfigurasi Firebase**
   - Buat project baru di [Firebase Console](https://console.firebase.google.com)
   - Download file konfigurasi (`google-services.json` untuk Android, `GoogleService-Info.plist` untuk iOS)
   - Letakkan file konfigurasi di direktori yang sesuai

4. **Jalankan Aplikasi**
   ```bash
   flutter run
   ```
